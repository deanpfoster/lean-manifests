# Cslib.Foundations.Semantics.LTS.Notation

## Module Summary

Provides the `create_lts` command and `lts` attribute for automatically generating LTS structures and transition notation (`[mu]->` and `[mus]->>`) from labelled transition relations.

## Vocabulary

| Name | Kind | Description |
|------|------|-------------|
| `create_lts` | command (meta) | Create an `LTS` from a labelled transition relation |
| `lts_transition_notation` | command (meta) | Add step/multi-step notation for an LTS |
| `lts_attr` | attribute (meta) | Register notation for an LTS via `@[lts ...]` |

## Theorems

None (meta commands only).
