# Cslib.Foundations.Semantics.LTS.Termination

## Module Summary

Defines `MayTerminate` (can reach a terminated state) and `Stuck` (not terminated and no transitions) for LTSs, parameterized by a `Terminated` predicate.

## Vocabulary

| Name | Kind | Description |
|------|------|-------------|
| `LTS.MayTerminate` | def | State can reach a terminated state |
| `LTS.Stuck` | def | State is not terminated and has no outgoing transitions |

## Theorems

None (definitions only).
